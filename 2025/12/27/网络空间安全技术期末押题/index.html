网络空间安全技术期末押题

发表于2025-12-27|更新于2025-12-27|scu网安专业课
|浏览量:

作业题

选择题

  1. 下列措施中，主要用于实现 “网络信息完整性保护” 的是（ ）

    A 使用密码算法对敏感数据进行加密传输，防止被未授权者窃取

    B 部署服务器集群，避免因单点设备故障导致网络服务中断

    C 对传输的重要文件计算哈希值，接收方校验哈希值是否一致

    D 要求用户登录时输入手机验证码，确认操作是本人发起

  2. 以下对于网络安全防护工作描述正确的是（ ）

    A 网络安全事件发生后，应优先聚焦紧急止损，事件处理完毕后无需投入资源开展预防工作

    B 需尽早发现网络系统中的安全隐患（如软件漏洞、异常访问行为）并及时处置，避免风险扩散

    C 网络安全防护只需依赖高端杀毒软件与防火墙，无需开展日常的安全监测

    D 面对网络安全风险，仅需保护核心业务数据，非核心系统的微小隐患可暂时忽视

  3. 根据《个人信息保护法》，处理个人信息时下列行为合法的是（ ）

    A 未经用户同意，向第三方共享其手机号、收货地址等信息

    B 为提升服务效率，默认收集用户的浏览记录、位置信息

    C 处理敏感个人信息（如生物识别、医疗健康数据）前，取得用户的单独同意

    D 因 “优化算法” 需要，长期存储已注销用户的全部个人信息

  4. 关于 “网络空间” 的定义，下列描述最准确的是（ ）

    A 仅指互联网所覆盖的虚拟通信环境，不包含任何实体设备

    B 指由计算机网络、网络设施、信息数据及用户交互行为构成的虚拟与现实结合的空间

    C 仅指存储和传输网络数据的服务器集群，与用户无关

    D 指各类社交平台、电商平台等互联网应用组成的虚拟社区

  5. 关于 “网络空间安全” 的核心保护范畴，下列描述错误的是（ ）

    A 需保护网络设施（如路由器、服务器）免受物理破坏或恶意入侵

    B 需保护网络中的数据（如用户隐私、业务数据）不被窃取或篡改

    C 需保障网络服务（如电商平台、政务系统）的持续可用，避免服务中断

    D 仅需保护互联网环境，无需关注企业内部局域网、物联网等非互联网场景

  6. 某企业发现服务器存在高危漏洞后，先评估漏洞被利用的概率、可能造成的业务损失，再制定补丁修复计划。这一过程最符合网络安全风险管理的哪一步骤 ( )

    A 风险识别 (发现漏洞)

    B 风险评估 (分析漏洞影响与概率)

    C 风险处置 (制定修复计划)

    D 风险监控 (跟踪漏洞修复进度)

  7. 某企业在修复服务器高危漏洞后，持续跟踪该服务器的访问日志、漏洞扫描结果，确认漏洞未被再次利用且无新风险产生。这一行为的核心目的是网络安全风险管理中的 ( )
    A 验证风险处置效果，确保风险已受控

    B 重新识别新的安全风险点

    C 评估未修复风险的潜在损失

    D 制定新的风险处置计划

  8. 根据风险管理定义,下列关于“信息安全风险管理”的核心特征描述,最符合上述定义的是( )

    A. 仅需通过技术手段控制已发生的风险事件,无需主动识别系统中潜在的不确定性因素

    B. 核心是先识别可能影响系统或组织的风险,再通过管理、控制等过程使风险消除、最小化或降至可接受水平

    C. 重点是消除组织机构内所有风险,确保不存在任何可能影响业务的不确定因素

    D. 只需在风险发生后优化应对措施,无需在日常工作中开展风险识别与管理

  9. 某公司采用 RBAC 模型管理办公系统权限，设定 “财务专员” 角色包含 “查询报销数据”“提交付款申请” 权限，“财务主管” 角色包含 “财务专员” 的所有权限 +“审批付款申请” 权限。下列说法正确的是 ( )
    A 若用户仅被分配 “财务专员” 角色，可自主添加 “审批付款申请” 权限

    B 若用户被分配 “财务主管” 角色，无需额外授权即可查询报销数据

    C “财务主管” 与 “财务专员” 属于 “角色互斥” 关系，同一用户不可同时拥有

    D 权限分配需直接针对单个用户，无法通过角色批量授权

  10. 网络安全中 “身份鉴别” 的核心目的是 ( )
    A 防止网络数据在传输过程中被篡改

    B 验证用户或设备声称的身份是否真实有效

    C 确保敏感信息仅被授权用户访问

    D 监控网络中所有用户的操作行为

  11. 某普通互联网公司希望实现 “权限分配灵活、可按部门／岗位批量授权，且管理员便于维护”，最适合采用的访问控制模型是 ( )
    A 自主访问控制 (DAC)

    B 强制访问控制 (MAC)

    C 基于角色的访问控制 (RBAC)

    D 基于属性的访问控制 (ABAC)

  12. 下列关于自主访问控制 (DAC) 和强制访问控制 (MAC) 的差异分析，说法错误的是 ( )
    A. DAC 的权限分配主体是资源所有者，MAC 的权限分配主体是系统管理员 (基于安全策略)

    B. DAC 中资源所有者可自主回收已授予的权限，MAC 中普通用户无权修改自身或他人的访问权限

    C. DAC 的安全粒度更细 (可针对单个文件授权给特定用户)，MAC 的安全粒度仅能按安全级别划分 (如 “一般 / 中级 / 高级”)

    D. DAC 适用于对安全性要求极高的防护场景，MAC 适用于普通企业的灵活授权场景

  13. 相比于传统的 “用户名 + 静态密码” 鉴别方式，采用 “智能卡 + 指纹验证” 的身份鉴别技术，核心安全优势是 ( )
    A 操作更便捷，无需记忆复杂信息

    B 鉴别因子更难被伪造或窃取，抗攻击能力更强

    C 适用场景更广泛，可覆盖所有网络系统

    D 部署成本更低，无需额外硬件支持

  14. 某企业的 NIDS 无法检测 HTTPS 流量中的恶意攻击，核心原因是？
    A NIDS 不支持 TCP 协议，而 HTTPS 基于 TCP 传输

    B 加密流量的载荷被 TLS 加密，NIDS 无法解析其内容

    C HTTPS 使用 443 端口，不在 NIDS 的默认监控端口范围内

    D NIDS 部署在防火墙之后，无法获取 HTTPS 流量

  15. 相比 “特征检测”，IDS 的 “异常检测” 最突出的优势是？
    A 误报率极低，可直接用于自动阻断攻击

    B 能检测未被特征库收录的零日攻击（0-day）

    C 对检测规则的维护成本远低于特征检测

    D 仅需分析单个数据包即可完成攻击判定

  16. 企业将 Web 服务器部署在 DMZ 区域，符合安全规范的防火墙规则是？
    A 允许 DMZ 的 Web 服务器主动连接内网数据库服务器

    B 允许外网任意 IP 访问 Web 服务器的所有端口（1-65535）

    C 允许内网终端访问 Web 服务器的 80/443 端口，同时禁止 DMZ 访问内网

    D 禁止记录 DMZ 与外网的流量日志，避免日志量过载

  17. 某企业的 IPS 频繁阻断合法的文件传输流量，最可能的原因是？
    A. IPS 采用旁路部署，无法实际干预流量

    B. IPS 特征库过旧，未收录该文件传输协议的规则

    C. IPS 的异常检测基线设置过严，将正常传输识别为异常

    D. IPS 的私钥泄露，导致加密流量解析错误

  18. IPsec 中 “抗重放攻击” 的实现机制是
    A 采用一次性会话密钥，每次连接生成新密钥，防止旧密钥被复用

    B 通过 IKE 主模式加密身份信息，避免身份被伪造后发起重放攻击

    C 为每个 ESP/AH 数据包分配唯一序列号，接收方通过窗口机制校验是否重复

    D 限制单位时间内的 IKE 协商次数，防止暴力重放协商请求

简答题

  1. 简述你对人工智能，云计算，大数据等新技术的安全问题的理解, 谈谈如何应对人工智能，云计算，大数据等新技术的安全问题。
  2. 简述网络空间安全四层模型，并讨论各层之间的关系。
  3. 请简述开放系统互连安全体系结构中安全机制与安全服务的联系与区别，并简述如何利用这些机制为川大网站提供安全服务。
  4. 请简述数字签名机制的特性和主要应用场景，讨论并举例说明如何利用数字签名机制保护数字资产。
  5. 请简述国产化替代对我国网络空间安全风险防范的作用并举例说明。
  6. 请简述PDR模型的主要思想并分析其与演变模型PPDR、PDRR的联系与区别
  7. 身份鉴别中常使用“多因子认证”。请说明多因子认证一般由哪几类因子组成，并举例说明每类因子的典型实现方式。
  8. 基于角色的访问控制（RBAC）常用于各类信息系统中。请阐述RBAC的核心概念（如角色、权限、继承等），并举例说明引入RBAC时可以解决哪些管理上的问题。
  9. 常见的防火墙部署方式有哪些。请画出不同部署方式的拓扑示意图，并分析其主要特点和区别。以“校园网出口”为例，讨论应采用那哪种部署方式并给出理由。
  10. 入侵检测系统（IDS）通常分为基于特征的检测和基于异常的检测两类。请阐述两类检测方法的核心思想；同时，请举一个网络攻击场景来说明两者可能的检测效果差异。

分析题

  1. 随着人工智能技术的不断进步，大型机器学习模型(大模型)在网络空间安全领域扮演着越来越重要的角色。它们在恶意软件检测、入侵检测系统、异常行为分析等方面展现出巨大的潜力。然而，大模型的广泛应用也带来了新的安全挑战。

    (1)描述并分析大型机器学习模型在网络空间安全中可能面临的主要安全风险。(4 分)
    (2)针对第一小问中提到的安全风险，讨论相应的安全对策。(5分)

  2. 今年“4·15”全民国家安全教育日期间，国家安全机关公开了一起境外某公司非法搜集窃取我稀土领域国家秘密案。该公司中方雇员叶某某在外籍员工指挥下，利诱收买国内某稀土公司副总经理成某，为境外刺探并非法提供涉及7项机密级国家秘密。 2023年11月，江西省南昌市中级人民法院分别为境外收买、非法提供国家秘密罪判处叶某某有期徒刑11年，以为境外非法提供国家秘密罪、受斯罪判处成某有期徒刑11年6个月。
    (1)国家保密局限定的安全隔离网闸使用环境有哪些？列举其中两个，可画图说明。(4 分)
    (2)在信息化和数字化高度发展的今天，网络空间安全技术如何应对国家层面面临的间谍行为和数据窃取威胁？请结合网络安全技术的具体手段，分析其在防范此类非法信息泄露中的应用。(5分)
